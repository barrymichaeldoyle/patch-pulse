  Installation takes about 30 seconds and is free. Patch Pulse needs at least 5
  active workspace installs before Slack will approve it for the official
  Marketplace. Installing now helps the project reach that threshold.
</Aside>

## How it works

Once installed, the bot monitors packages you care about and sends a Slack notification when a new version is published to the npm registry.
